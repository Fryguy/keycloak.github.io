Installing Distribution Files

The Keycloak Server is contained in one distribution file:

  • 'keycloak-2.4.0.Final.[zip|tar.gz]'

The 'keycloak-2.4.0.Final.[zip|tar.gz]' file is the server only distribution. It contains nothing other than the scripts and binaries to run the Keycloak server.

To unpack these files run the unzip or gunzip and tar utilities.
